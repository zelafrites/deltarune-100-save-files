# DELTARUNE 100% Save Files

Fully completed **DELTARUNE save files** (up until the save point)<br>

## Download

- Download any save file you want

## Installation Guide

1. Open the Run panel with Super + R
2. Type %appdata%
3. You'll be put in roaming most likely, make sure you go to **/AppData/Local/DELTARUNE**

## Replacing Save Files

1. Rename your boss fight file to "filech<1-4>_0/1/2
- (0 is first slot, 1 is second, 2 is third)
2. Replace the existing file in the folder (if there even is one)

## Customizing Your Save

1. Open the save file with anything you want, I recommend notepad or notepad++
2. At the top, you'll see:

**DAVID**<br>
**KRIS**

**(bunch of numbers)**

3. Edit "DAVID" to your name and "KRIS" to your vessel's name

## Done!

Launch the game and enjoy your new save file

## Notes

- Always back up your original saves before replacing them
- Make sure file names match exactly, or the game will NOT detect them
